### 查看机器使用情况

```sh
# 查看内存（内存总量+内存使用量+内存使用率）
# free -k：以kb为单位，查看内存使使用情况
# sed -n '2p'：流编辑器，只输出第二行（输出那些匹配的行）
# awk '{print $2}'：每行按空格或TAB分割，输出文本中的第二项
free -k | sed -n '2p' | awk '{print "total: "$2" KB, used: "$3" KB,used percent: "$3*100/$2"%"}'
```
